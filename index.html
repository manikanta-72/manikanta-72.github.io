# github.com/manikanta-72/multi-image-generation

Multi-Image Generation using Cycle GAN

Augmented CycleGAN to learn many-to-many domain mappings using noise as a latent variable. Injecting different noise values produces different versions of the target image from a single source.

Evaluated on Edges2Shoes, Night2Day, CelebA, and FERDB datasets.

cycle_gan.pyPython · PyTorch
class AugmentedCycleGAN(nn.Module):
    def __init__(self, latent_dim=8):
        super().__init__()
        self.gen_AB = Generator(latent_dim)   # A → B
        self.gen_BA = Generator(latent_dim)   # B → A
        self.disc_A = Discriminator()
        self.disc_B = Discriminator()

    def forward(self, x_A, z):
        """Many-to-many mapping via noise latent z."""
        x_AB  = self.gen_AB(x_A, z)   # translate + condition on z
        x_ABA = self.gen_BA(x_AB, z)  # reconstruct
        return x_AB, x_ABA
# projects/dist_chat.java  ·  Jan – Mar 2023

Distributed Multi-Room Chat Application

Multi-room chat app deployed on AWS EC2. Pub/sub architecture using Apache Kafka as messaging middleware; partitioned topics per room ensure message ordering and horizontal scalability.

Stack: Java Spring Boot · React · Apache Kafka · AWS EC2

ChatConsumer.javaJava · Spring Boot
// Kafka consumer: one listener per chat room
@KafkaListener(topics = "#{roomId}", groupId = "chat-group")
public void consume(String message,
                      @Header("roomId") String roomId) {
    ChatMessage msg = mapper.readValue(message, ChatMessage.class);
    webSocket.broadcast(roomId, msg);
}
